# UI Guidelines

## TODO App Interface Standards

### Task List Item Layout

- Each task is displayed as a checklist-style component.
- The **checkbox** is positioned on the **left side** of the task text.
  - The checkmark must be **thick, bold, and black** when checked.
- Task text uses **black font**.

### Actions

- The **delete action** is represented by a small **trash can icon** to clearly communicate its destructive intent. No text label is required.
